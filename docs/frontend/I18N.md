# I18N

malziSPACE uses a small browser-side i18n layer so the public site can stay static, CSP-safe, and easy to ship without a bundler.

## Structure

- `apps/web/public/assets/i18n.js`: translation engine + shared/common + space/app strings
- `apps/web/public/assets/site-i18n-dicts.js`: landing/legal/site dictionaries
- `apps/web/public/space.html`: product UI with `data-i18n*` attributes
- `apps/web/public/index.html`, `apps/web/public/privacy.html`, `apps/web/public/agb.html`, `apps/web/public/impressum.html`: site/legal pages using the site dictionary layer

## Supported locale selection

Locale resolution order:

1. `?lang=<code>` query param
2. `<html lang="...">`
3. browser language
4. fallback to `de`

Current bundled locales:

- `de`
- `en`

## Supported bindings

The engine supports these attributes:

- `data-i18n`: replace `textContent`
- `data-i18n-html`: replace `innerHTML` for trusted static copy only
- `data-i18n-placeholder`: replace `placeholder` and `data-placeholder`
- `data-i18n-title`: replace `title`
- `data-i18n-aria-label`: replace `aria-label`
- `data-i18n-data-label`: replace `data-label`

## Runtime strings

Static HTML is not enough. Client-side code must route visible strings through `window.MZ_I18N.t(...)`.

Current runtime-covered files:

- `apps/web/public/assets/modules/` (modular editor)
- `apps/web/public/assets/index-create.js`

Rule: any new user-visible string added in JS must use a translation key, not an inline literal.

## Legal content rule

Legal content is translated for usability, but privacy and terms must still be reviewed whenever:

- Firebase/Google services change
- regions change
- a new tracking/telemetry/security service is introduced
- retention/storage behavior changes

Keep `docs/legal/PRIVACY_STACK.md` and `apps/web/public/privacy.html` aligned.

## Tests

Local verification includes browser checks for translation and legal pages:

```bash
npm run test:e2e:i18n
```

This validates:

- English landing page rendering
- English space UI rendering
- privacy page service/region facts
- translated terms page
- translated imprint page
